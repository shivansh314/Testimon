# 🎤 TestiMon – Customer Testimonial Collection Platform

## 🧩 About  
TestiMon is a full-stack MERN application designed to help businesses collect, manage, and showcase authentic customer testimonials. It supports video and text reviews, features a centralized dashboard, and integrates with other platforms so you can display glowing feedback anywhere on the web.


## 🚀 Features

-  Collect video and text-based testimonials from users
-  Sentiment analysis on submitted reviews to gauge customer satisfaction
- Dashboard to view and manage all testimonials in one place
-  Embed testimonials into other websites with ease
-  Admin panel to approve, reject, or feature testimonials
-  Secure backend and user management
-  Search and filter testimonials by sentiment or keyword

---

## 🧰 Tech Stack

- **Frontend:** React, Tailwind CSS, Redux  
- **Backend:** Node.js, Express.js, MongoDB  

---

## 🛠️ Installation Instructions

### 📦 Prerequisites
- Node.js & npm  
- MongoDB  

---

### 🔧 Frontend Setup

```bash
cd frontend
npm install         # Install dependencies
npm run dev         # Start the frontend development server
```
### 🔧 Backend Setup
```bash
cd backend
npm install         # Install dependencies
npm run dev         # Start the backend development server```
